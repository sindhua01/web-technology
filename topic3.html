WWW and Web 2.0

*World Wide Web (WWW)*

The World Wide Web (WWW) is a system of interlinked hypertext documents that can be accessed via the internet.
It was invented by Tim Berners-Lee in 1989.

*Key Features:*

1. *Hypertext*:

Documents linked together using hyperlinks.

2. *Web Pages*:

Documents displayed on the web, containing text, images, and multimedia.
<>3. *Web Browsers*: Software applications used to access and display web pages (e.g., Google Chrome, Mozilla Firefox).

4.*HTTP*:

Hypertext Transfer Protocol, the communication protocol used to transfer data between web servers and browsers.

*Web 2.0*

Web 2.0 refers to the second generation of the World Wide Web, characterized by greater user interaction,
collaboration, and sharing of user-generated content.*Key Features:*

*User-Generated Content*:

Users create and share content, such as blogs, videos, and social media posts.

2. *Social Media*:

platforms that enable users to interact, share, and collaborate
(e.g., Facebook, Twitter, YouTube).

3. *Ajax*:

Asynchronous JavaScript and XML, a programming technique that enables dynamic web page updates without full page reloads.

4. *Cloud Computing*:

On-demand access to computing resources, such as storage, processing power, and applications, over the internet.

*Characteristics:*

1. *Participatory*:

Users actively contribute and participate in online communities.

2. *Interactive*:

Web applications enable real-time interaction and feedback.

3. *Dynamic*:

Web content is constantly updated and refreshed.

4. *Collaborative*:

Users work together on projects, share ideas, and build communities.
